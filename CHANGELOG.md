# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning.

## [0.1.0] - 2026-04-22
### Added
- Initial repository scaffold for the OpsToutatis module.
- Module manifest and recursive loader for `src/Public` and `src/Private`.
- Bootstrap installers for Windows (`scripts/install.ps1`) and Linux (`scripts/install.sh`).
- Cross-platform GitHub Actions workflow running Pester on Windows and Ubuntu.
- Initial project documentation and contribution guide.
